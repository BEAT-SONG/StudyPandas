<데이터보기>
- 데이터 구조를 이해하는 기술
- 판다스에서 가장 많이 사용되는 데이터 구조

- 데이터 프레임 = 행과 열로 구성된 데이터

- 판다스 = 데이터 기반, 넌파일을 포함된 범위, 일반적으로 데이터를 분석할 때 판다스를 사용
- 넌파일 = 데이터 기반, 0과 1에 숫자로 이루어진 행렬 등 수학적인 개념에 강력함

<목차>
- 판다스 데이터 구조
- 시리즈와 데이터프레임
+ 시리즈(Series) = 하나의 컬럼(속성을 제외한)의 데이터, 1차원 자료구조
+ 데이터프레임(DataFrame) = 여러개의 시리즈, 2차원 자료구조
- 데이터프레임 조작
- 단일변수 데이터 보기
+ 특정 데이터 출력
- 데이터 병합과 연결

<시리즈>
- 1차원 레이블(테이블)이 지정된 배열
+ 배열 == list(리스트)
- 변수명(속성, 퓨쳐, 컬럼명) 하나에 대한 데이터들
- 열 레이블을 총칭
